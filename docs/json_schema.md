# GitPulseMLC JSON Output Specification

This document describes the structure of the JSON output generated by `gitpulse --json`.
You can also find the formal JSON Schema in [schema.json](./schema.json).

## Root Object

| Field | Type | Description |
| :--- | :--- | :--- |
| `version` | string | The version of GitPulseMLC that generated the report. |
| `summary` | object | Global statistics across all scanned repositories. |
| `results` | array | List of `RepoStatus` objects, one for each project. |

---

## Summary Object

| Field | Type | Description |
| :--- | :--- | :--- |
| `total` | int | Total number of projects scanned. |
| `dirty` | int | Number of repositories with uncommitted changes. |
| `ahead` | int | Number of repositories with local commits not yet pushed. |
| `behind` | int | Number of repositories with remote commits not yet pulled. |
| `errors` | int | Number of projects that failed to scan (e.g., directory missing). |

---

## RepoStatus Object

| Field | Type | Description |
| :--- | :--- | :--- |
| `name` | string | The display name of the project. |
| `path` | string | The absolute filesystem path (or shortened path if replacement settings are used). |
| `current_branch` | string | Name of the currently checked-out branch. |
| `last_commit_id` | string | SHA-1 hash of the latest commit on HEAD. |
| `last_commit_msg`| string | Subject line of the latest commit. |
| `last_commit_at` | string | ISO 8601 timestamp of the latest commit. |
| `is_clean` | boolean | `true` if there are no uncommitted changes. |
| `modified_count` | int | Total count of modified files. |
| `stash_count` | int | Number of entries in the git stash. |
| `changed_files` | object | Map of file extensions to counts (e.g., `".go": 3`). |
| `ahead` | int | Commits ahead of the tracking remote branch. |
| `behind` | int | Commits behind the tracking remote branch. |
| `has_no_remote` | boolean | `true` if the repository has no configured remotes. |
| `local_branches` | array | List of `BranchInfo` objects for all local branches. |
| `error` | string | Error message if the scan failed (only present on error). |

---

## BranchInfo Object

| Field | Type | Description |
| :--- | :--- | :--- |
| `name` | string | Name of the local branch. |
| `is_head` | boolean | `true` if this is the active branch. |
| `ahead` | int | Commits ahead of remote. |
| `behind` | int | Commits behind remote. |
| `last_commit_at` | string | ISO 8601 timestamp of the last commit on this branch. |
| `is_stale` | boolean | `true` if the branch is older than 30 days (excluding main/master). |

---

## Example Output

```json
{
  "version": "1.1.0",
  "summary": {
    "total": 1,
    "dirty": 1,
    "ahead": 0,
    "behind": 0,
    "errors": 0
  },
  "results": [
    {
      "name": "my-project",
      "path": "@/dev/my-project",
      "current_branch": "feature/ui",
      "is_clean": false,
      "changed_files": {
        ".go": 2,
        ".md": 1
      },
      "local_branches": [
        {
          "name": "main",
          "is_head": false,
          "ahead": 0,
          "behind": 0,
          "is_stale": false
        }
      ]
    }
  ]
}
```
